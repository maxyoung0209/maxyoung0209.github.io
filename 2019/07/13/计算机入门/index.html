计算机科学与技术必读书目（计算机原理篇）

发表于 2019-07-13   |   分类于 计算机学习

计算机科学与技术是国家一级学科，下设信息安全、软件工程、计算机软件与理论、计算机系统结构、计算机应用技术、计算机技术等专业。

本文罗列了一些关于计算机原理的书，这些书通俗易懂，适合自学。

计算机科学与技术专业应该如何学习？有人建议自顶向下学习，即先学习比较新的、应用型的技术和理论，然后不断往底层探索学习。也有人建议自下而上学习，即沿着计算机发展线一步一步学习。我认为这两种学习方式都比较科学，但第一种方式更适合于“半路出家”或者自学者，第二种方式适合当下时间比较充裕没有生存压力的学生。但无论怎样，要切忌造空中楼阁，对计算机系统没有一个全面的认识是很难持续发展的。

下面列一些必读经典书籍

看这本书的目录，它更像一本科普书。

作者沿着历史长河介绍了早期通信模型，以及包括莫尔斯电码、布莱叶盲文在内的编码方式。介绍了继电器与二进制的相似性，逻辑学如何从文字描述时代跨入数学描述时代的，逻辑学又是如何与开关电路联系起来的。以及利用开关电路构建的一些电路模型，介绍了冯诺依曼架构的计算机模型，介绍了计算机是如何通过指令进行自动化操作的。然后通过一层层抽象，计算机是如何发展到现代的。

3、《大话计算机：计算机系统底层架构原理极限剖析》

目录：

第1章介绍数字计算机的设计思路，制作一个按键计算器，在这个过程中逐步理解数字计算机底层原理。
第2章在第1章的基础上，改造按键计算器，实现能够按照编好的程序自动计算，并介绍对应的处理器内部架构概念。
第3章介绍电子计算机的发展史，包括芯片制造等内容。
第4章介绍流水线相关知识，包括流水线、分支预测、乱序执行、超标量等内容。
第5章介绍计算机程序架构，理解单个、多个程序如何在处理器上编译、链接并最终运行的过程。
第6章介绍缓存以及多处理器并行执行系统的体系结构，包括互联架构、缓存一致性架构的原理和实现。
第7章介绍计算机I/O基本原理，包括PCIE、USB、SAS三大I/O体系。
第8章介绍计算机是如何处理声音和图像的，包括3D渲染和图形加速原理架构和实现。第9章介绍大规模并行计算、超级计算机原理和架构，以及可编程逻辑器件（如FPGA等）的原理和架构。
第10章介绍现代计算机操作系统基本原理和架构，包括内存管理、任务调度、中断管理、时间管理等架构原理。
第11章介绍现代计算机形态和生态体系，包括计算、网络、存储方面的实际计算机产品和生态。
第12章介绍机器学习和人工智能底层原理和架构实现。

这套书分为三册，内容非常多，比上面两本都要专业，我买了第一版，说实话挺难啃的，但读得很快乐很仔细，毕竟整个计算机的历史大都在里面了。这套书值得计算机科学与技术专业学生用四年读透它。

这本书主要介绍系统软件的运行机制和原理，涉及在Windows和Linux两个系统平台上，一个应用程序在编译、链接和运行时刻所发生的各种事项，包括：代码指令是如何保存的，库文件如何与应用程序代码静态链接，应用程序如何被装载到内存中并开始运行，动态链接如何实现，C/C++运行库的工作原理，以及操作系统提供的系统服务是如何被调用的。每个技术专题都配备了大量图、表和代码实例，力求将复杂的机制以简洁的形式表达出来。本书最后还提供了一个小巧且跨平台的C/C++运行库MiniCRT，综合展示了与运行库相关的各种技术。
